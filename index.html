System Capabilities & Development Status

Live Demo

Core Temperature Monitoring

Dual DS18B20 sensor array providing temperature measurements. Real-time thermal state assessment with breed-specific reference ranges.

Live Demo

Photoplethysmography (PPG)

Optical heart rate detection using reflective infrared photoplethysmography. Continuous cardiac rhythm monitoring at 100Hz sampling frequency.

Live Demo

Tri-Axial Accelerometry

MPU9250 9-DOF IMU providing 3-axis acceleration data for activity classification and posture detection. Enables automated rest/active state determination.

In Development

Respiratory Rate Analysis

Capacitive stretch sensor integration for non-invasive respiratory monitoring. Signal processing algorithms under development for breath-per-minute calculation from thoracic expansion patterns.

Future Feature

Automated Level of Activity Classification

Machine learning classifier to automatically detect activity states (resting, walking, running, playing) based on multi-sensor fusion. Currently requires manual mode selection.

Future Feature

Longitudinal Data Analytics

Cloud-based data aggregation and trend analysis. Historical baselines for personalized health anomaly detection and veterinary reporting integration.
